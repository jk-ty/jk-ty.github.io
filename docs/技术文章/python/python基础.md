## 4.列表与元组

#### 列表

```
a[1:6]	#获取第2到第6个元素    左开右闭区间

a[1:6:2]	#获取第2，第4和第6个元素

a.append(b) 	向a列表中添加元素b

a.remove(b)	删除指定a列表中的值b

a.count(b)	统计b在a列表中出现的次数

a.shot()	对列表进行排序
```
